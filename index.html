Lines命令一览

-- Level 0 --

help 查看Lines的帮助

cls 清空Lines的屏幕

quit 退出Lines

net [code] 链接到密钥code的计算机并进行操作

-- Level 1 --

systeminfo 查看对方计算机的信息

username 查看对方当前用户名

computername 查看对方当前计算机名

ver 显示对方计算机版本

ipconfig 查看对方计算机的网络信息

disks 查看对方所有硬盘盘符

dir [path] 查看对方盘符路径为path的所有文件

-- Level 2 --

shutdown 关闭对方计算机

restart 重启对方计算机

msg [words] 在对方的计算机上弹出对话框

report [words] 在对方计算机上念出文字

fun 在对方计算机C:\下运行在本机的脚本(.hta/.bat/.cmd/.exe/.com)

start [files] 在对方电脑下打开或运行完全目录为files的文件或程序

-- Level 3 --

netuser [password] 重新设定对方计算机当前用户的密码为password

format [volume] 强制静默格式化对方计算机盘符为volume的硬盘

del [files] 删除对方计算机下完全路径为files的文件

taskkill [programname] 关闭进程名为programname的程序

-- Note --

report命令需要对方计算机支持TTS

Level 0的命令在本机Lines生效，其余命令会传输到对方计算机的Lines端执行并收到反馈

如果语法错误或不符合对方计算机环境或命令不存在则无法收到反馈

需要保证对方计算机开机并网络环境畅通才能控制

如果没有连接到任何计算机，那么Level 1~3的命令无效
